# Grizz-POP

An event organizer website designed for Oakland University students. Users can create, share, and leave reviews on campus events. Includes a dedicated **SetupCrew** interface for event management.

## Features

- **Event Creation** – Students can create and share events
- **RSVP System** – Users can reserve spots for events
- **Event Reviews** – Leave and read reviews on past events
- **Pin Events** – Save favorite events for quick access
- **SetupCrew Interface** – Special dashboard for event administrators to manage events
- **Cancel/Edit Events** – Modify or cancel existing events
- **Messaging System** – Communicate between event organizers and attendees

## Technologies Used

- PHP
- MySQL (SQL database)
- HTML5
- CSS3
- JavaScript

## Database Structure

Includes tables for:
- User management
- Event reservations (RSVP)
- Pinned events
- Event status tracking
